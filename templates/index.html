Stock Price Prediction API

This API provides stock data and price predictions.

Get Stock Data

GET /api/stock/<symbol>

Example: /api/stock/AAPL

Predict Stock Prices

POST /api/predict

Body: {"symbol": "AAPL", "days_ahead": 7}

Suggest Symbols

GET /api/suggest/<partial>

Example: /api/suggest/app

Market Status

GET /api/market-status
